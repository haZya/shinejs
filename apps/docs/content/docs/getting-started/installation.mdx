---
title: Installation
description: Install shinejs and choose your integration path.
---

Use your preferred package manager.

```bash
npm install shinejs
```

```bash
pnpm add shinejs
```

```bash
yarn add shinejs
```

## Import paths

- Core class API: `shinejs`
- React hook API: `shinejs/react`

```ts
import { Shine } from "@hazya/shinejs";
import { useShine } from "@hazya/shinejs/react";
```

## Requirements

- Modern browser environment.
- ESM-compatible tooling.
- React + Next.js optional (only for [`useShine`](/docs/guides/react-use-shine)).
