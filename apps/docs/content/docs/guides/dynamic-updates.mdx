---
title: Dynamic Updates
description: Update content, color, and shadow parameters at runtime.
---

Use `update()` for live changes.

## Update content text

### Overview

<DynamicUpdateContentPreview />

### Usage

```ts
update({ content: "Hello World" });
```

## Update config values

### Overview

<DynamicUpdateConfigPreview />

### Usage

```ts
update({
  config: {
    blur: 48,
    opacity: 0.22,
    offset: 0.2,
  },
});
```

## Update light behavior

### Overview

<DynamicUpdateLightPreview />

### Usage

```ts
update({
  light: {
    intensity: 2,
    position: { x: 320, y: 140 },
  },
});
```

## Color picker flow (`shadowRGB`)

### Overview

<DynamicUpdateColorPreview />

### Usage

```ts
update({
  config: {
    shadowRGB: { r: 26, g: 54, b: 93 },
  },
});
```
