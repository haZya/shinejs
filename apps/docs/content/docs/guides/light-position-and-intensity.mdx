---
title: Light Position and Intensity
description: Follow the mouse, set fixed positions, or animate custom paths.
---

`light.position` accepts either `"followMouse"` or a fixed point. For auto-pilot, animate a fixed `{x, y}` path each frame.

## Auto-pilot animation

### Overview

<LightPositionAutoPilotPreview />

### Usage

<PreviewCodeTabs
  codes={[
    {
      title: "Shine Component",
      language: "tsx",
      code: `
  const [lightPosition, setLightPosition] = useState({ x: 0, y: 0 });

  <Shine as="h2" options={{ light: { position: lightPosition } }}>
    Auto Pilot
  </Shine>;

  const t = Date.now() * 0.00025 * Math.PI * 2;
  setLightPosition({
    x: window.innerWidth * 0.5 + window.innerWidth * 0.4 * Math.cos(t),
    y: window.innerHeight * 0.5 + window.innerHeight * 0.4 * Math.sin(t * 0.7),
  });`,
    },
    {
      title: "useShine Hook",
      language: "ts",
      code: `
  const animate = () => {
    const t = Date.now() * 0.00025 * Math.PI * 2;
    update({
      light: {
        position: {
          x: window.innerWidth * 0.5 + window.innerWidth * 0.4 * Math.cos(t),
          y: window.innerHeight * 0.5 + window.innerHeight * 0.4 * Math.sin(t * 0.7),
        },
      },
    });

    requestAnimationFrame(animate);
  };

  requestAnimationFrame(animate);`,
    },
    {
      title: "Shine Class",
      language: "ts",
      code: `
  const animate = () => {
    const t = Date.now() * 0.00025 * Math.PI * 2;
    shine.update({
      light: {
        position: {
          x: window.innerWidth * 0.5 + window.innerWidth * 0.4 * Math.cos(t),
          y: window.innerHeight * 0.5 + window.innerHeight * 0.4 * Math.sin(t * 0.7),
        },
      },
    });

    requestAnimationFrame(animate);
  };

  requestAnimationFrame(animate);`,
    },
  ]}
/>
