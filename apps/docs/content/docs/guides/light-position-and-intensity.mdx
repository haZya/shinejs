---
title: Light Position and Intensity
description: Follow the mouse, set fixed positions, or animate custom paths.
---

`light.position` accepts either `"followMouse"` or a fixed point.

## Follow mouse

```ts
update({ light: { position: "followMouse" } });
```

## Fixed point

```ts
update({ light: { position: { x: 480, y: 220 } } });
```

## Intensity

```ts
update({ light: { intensity: 1.5 } });
```

Higher intensity amplifies perceived depth. Start around `1.0` and increase gradually.

## Auto-pilot animation

Use `requestAnimationFrame` and set `{x, y}` each frame for cinematic sweep effects.
