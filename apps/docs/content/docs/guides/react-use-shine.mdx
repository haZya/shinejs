## Example

`useShine(ref, options)` creates and manages a `Shine` instance for your element.

### Overview

### Basic usage

```tsx
"use client";

import { useRef } from "react";
import { useShine } from "@hazya/shinejs/react";

export function InteractiveTitle() {
  const ref = useRef<HTMLHeadingElement>(null);

  const { update } = useShine(ref, {
    light: { position: "followMouse" },
    config: { opacity: 0.24 },
  });

  return (
    <button onClick={() => update({ config: { opacity: 0.4 } })}>
      <h1 ref={ref}>Adjust Me</h1>
    </button>
  );
}
```

## Returned values

- `shine`: The underlying `Shine` instance (or `null` before mount).
- `update(newOptions)`: Merge updates at runtime without re-creating your component tree.

## Important behavior

- The hook initializes once the `ref.current` element exists.
- When the hook unmounts, it automatically calls `destroy()`.
- Config changes are diffed and applied with `update(...)` on the same instance.
