Welcome to the official `shinejs` documentation.

Use this site as both a learning path and a reference:

- Start with installation and quick starts.
- Follow guides for React, class API, and lifecycle/performance details.
- Copy examples for all major use cases from the demo app.
- Use API reference pages for exact option and method behavior.

## What you can build

- Cursor-aware text shadows for hero headlines.
- Animated "auto-pilot" light paths.
- Neumorphic-style card and image shadows.
- Dynamic, user-controlled shadow tuning with live sliders.
- Programmatic color palettes with `shadowRGB`.

## Coverage promise

This documentation covers every public export and all current demo scenarios:

- React hook usage (`useShine`).
- Direct class usage (`Shine`, `Color`).
- Text mode and children mode.
- Runtime updates (`update`, `updateContent`, light changes).
- Cleanup and performance behavior.
