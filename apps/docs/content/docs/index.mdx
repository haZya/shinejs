## Introduction

Welcome to the official `shinejs` documentation.

Use this site as both a learning path and a reference:

- Start with [installation](/docs/getting-started/installation), then choose a quick start.
- Learn the main usage patterns in the [guides](/docs/guides/react-use-shine).
- Explore practical scenarios in the [examples](/docs/examples/mouse-follow).
- Use the [API reference](/docs/api/shine-class) for exact option and method behavior.

## What you can build

- Cursor-aware text shadows for hero headlines.
- Animated auto-pilot light paths.
- Neumorphic-style card and image shadows.
- Dynamic, user-controlled shadow tuning with live sliders.
- Programmatic color palettes with `shadowRGB`.
